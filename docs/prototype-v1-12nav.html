📖 数据字典 + 历史查询模式 综合分析：

字段：apr_score — APR 通过率得分
类型：DECIMAL(5,4)   范围：0.0000 ~ 1.0000
含义：单台设备当日 APR 测试的通过比率，1.0 = 全部通过
计算逻辑：通过用例数 / 总执行用例数
常用换算：apr_score * 100 → 百分比形式（如 0.968 → 96.8%）
注意：NULL 表示当日无测试数据，非 0 分

📊 历史查询统计：该字段在 89% 的场景中会乘以 100 转百分比显示，建议统一使用 round(apr_score * 100, 1)。

SELECT a.test_date AS day, a.device_model, a.framework, round(avg(a.apr_score) * 100, 1) AS apr_rate_pct, round(avg(t.max_temp), 1) AS avg_max_temp, round(avg(t.cpu_freq_mhz), 0) AS avg_cpu_freq, CASE WHEN avg(a.apr_score)*100 < 90 THEN '异常' ELSE '正常' END AS status FROM dws.stability_apr_daily a LEFT JOIN dws.thermal_test_daily t ON a.device_model = t.device_model AND a.test_date = t.test_date WHERE a.test_date >= current_date - INTERVAL '7 days' GROUP BY 1, 2, 3 ORDER BY day ASC, device_model, framework;
